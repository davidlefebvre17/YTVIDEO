## Les 3 phases d'un conflit sur les marchés

### Phase 1 — Choc initial (heures/jours)
- Gap à l'ouverture sur devises refuges (CHF, USD, JPY), or, pétrole.
- Crypto chute puis rebondit rapidement (marché 24/7, premier à pricer).
- Le marché SURRÉAGIT souvent : le gap est fréquemment comblé dans les heures qui suivent.
- **Piège classique** : acheter la panique à l'ouverture -> stop loss touché quand le gap se comble. "Se précipiter bêtement à l'ouverture, c'est le meilleur moyen de prendre cher."

### Phase 2 — Pricing du risque réel (jours/semaines)
- Le marché évalue : conflit régional contenu ou menace systémique ?
- Critères d'escalade : frontière commune ? troupes au sol ? impact supply chain ? grandes puissances impliquées ?
- Les assets réagissent tant que l'incertitude dure, puis se stabilisent.

### Phase 3 — Normalisation ou escalade
- **Désescalade** : retour rapide des prix, gap comblé, refuges rendent leurs gains. OPPORTUNITÉ de trader l'inverse.
- **Escalade confirmée** : nouvelle jambe de stress, VIX monte, refuges accélèrent. Pivot vers fil conducteur géopolitique.

### Règle clé
Un gros conflit se prépare, il n'arrive pas de nulle part. Les conflits majeurs sont TOUJOURS précédés de signaux : accumulation militaire, déploiement naval, rhétorique diplomatique. Suivre les porte-avions, les bases militaires, les images satellites = indicateurs avancés.
