## Russie-Ukraine — Narration selon les phases

| Phase | Impact marché | Narration |
|-------|-------------|-----------|
| Status quo / guerre de position | Neutre (price in) | "Le conflit est dans les prix — le marché ne réagit plus aux titres quotidiens" |
| Escalade (nouvelles armes, offensive) | Risk-off EU, défense hausse | "L'escalade relance les craintes énergétiques en Europe — le gaz réagit" |
| Négociations / cessez-le-feu | Risk-on EU, défense baisse | "Un espoir de paix — l'Europe respire, les défensives rendent leurs gains" |
| Sanctions nouvelles | Impact sectoriel ciblé | "Les nouvelles sanctions ciblent [X] — ça touche directement les entreprises exposées" |

Le narrateur doit d'abord identifier dans quelle phase se situe le conflit avant de choisir son angle. En phase de status quo, ne pas forcer le sujet — le mentionner en contexte seulement si un asset spécifique réagit. En phase d'escalade ou de négociations, le conflit peut devenir le fil conducteur de l'épisode.
