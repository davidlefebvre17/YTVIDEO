## Taux d'interet : LE facteur numero un du forex

Les taux d'interet sont LE facteur qui fait bouger les devises. Pas les graphiques, pas les patterns — les taux.

### Mecanismes cles

- **Differentiel de taux = direction de la paire** : les capitaux vont la ou le rendement est le plus eleve. Si la Fed est a 5% et la BCE a 3%, les flux vont vers le dollar.
- **Ce ne sont pas les taux actuels qui comptent, mais les ANTICIPATIONS** : le marche price le futur. Quand le marche anticipe que la Fed va baisser ses taux, le dollar baisse AVANT la baisse effective.
- **Divergence de politique monetaire = tendance forte** : quand la Fed monte et la BCE attend, c'est massivement USD haussier.
- **Convergence = range** : quand les deux banques centrales font la meme chose, la paire range.

### Taux US — impacts cross-market

- **Taux montent -> Actions tech baissent** : les valorisations tech dependent du taux d'actualisation des flux futurs. Les valeurs de croissance sont les plus sensibles aux taux.
- **Taux montent -> Or baisse** : les obligations offrent un rendement, l'or non.
- **Taux montent -> Dollar monte** : rendement plus attractif attire les capitaux.
- **Courbe s'aplatit -> Banques souffrent** : le modele bancaire depend du spread de taux.

### Application narration

- "Deux banques centrales qui divergent = une tendance forex claire."
- "Les taux US attirent les capitaux vers le dollar."
- "La courbe s'aplatit et les bancaires souffrent."
