## Pourquoi les marches peuvent monter quand l'economie souffre

Cinq mecanismes expliquent ce paradoxe apparent :

1. **Concentration** : quelques mega-caps (FAANG/Mag7) tirent les indices. Le S&P 500 peut monter alors que la majorite des secteurs (energie, immobilier, banques, industrie) sont en rouge. Toujours regarder la breadth (largeur de marche).

2. **Forward looking** : les marches pricent le FUTUR, pas le present. En avril 2020 (plein Covid), les marches repartaient deja car les investisseurs anticipaient le vaccin et la reprise.

3. **TINA (There Is No Alternative)** : quand les taux sont proches de zero, les obligations rapportent rien -> l'argent afflue vers les actions par defaut.

4. **Reduction des societes cotees** : de plus en plus d'entreprises restent privees. Les faillites du secteur prive n'apparaissent pas dans les indices.

5. **Stimulation monetaire** : QE + taux bas = liquidite abondante -> gonfle les actifs financiers meme si l'economie reelle souffre.

Narration type : "Les indices sont en hausse mais attention — c'est surtout porte par une poignee de valeurs technologiques. Le tableau est plus nuance quand on regarde secteur par secteur."
